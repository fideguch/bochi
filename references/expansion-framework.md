# SCAMPER Idea Expansion Framework

Generate expansion ideas using 7 perspectives.
Select the 2-3 most effective for the user's idea. Do NOT present all 7.

## S: Substitute (代替)
「XXの代わりにYYを使ったらどうなるゆ？」
- Technology, target, channel, business model substitution

## C: Combine (結合)
「XXとYYを組み合わせたらどうなるゆ？」
- Cross-industry idea fusion, existing service integration

## A: Adapt (適応)
「別の業界/市場ではこれがどう使われてるゆ？」
- Successful pattern transfer from other industries

## M: Modify (変更)
「規模を10倍にしたら？逆に1/10にしたら？」
- Scale change, target change, pricing change

## P: Put to another use (転用)
「このアイデアの副産物は他に使えないゆ？」
- Data reuse, technology spinoff

## E: Eliminate (排除)
「何を取り除いたらもっとシンプルになるゆ？」
- Feature reduction, step elimination, cost simplification

## R: Reverse (逆転)
「逆の立場/逆の順序だったらどうなるゆ？」
- B2B→B2C, paid→free, push→pull

## Process
1. Summarize user's idea in one sentence
2. Select 2-3 most effective perspectives
3. Present one concrete expansion per perspective
4. User selects preferred direction → feed into Phase C research

## Edge Cases

- **全SCAMPER視点を拒否** → 「今の方向で深掘りするゆ？」でPhase C直行
- **ユーザー無反応** → 最も関連性の高い1視点を自動選択してPhase C進行
