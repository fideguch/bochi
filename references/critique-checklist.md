# Phase D Critique Checklist (HARD-GATE)

All checks must pass before proceeding to Phase E.

## 1. Source Quality Re-verification
- [ ] All sources have E-E-A-T score >= 28/40
- [ ] Failed sources excluded and replacement search executed

## 2. Alternative Approach Verification
- [ ] List 3 alternative approaches to the current recommendation
- [ ] State 1-line pros/cons for each alternative
- [ ] Confirm current approach is still the strongest; if not, return to Phase C

## 3. Information Freshness
- [ ] Primary sources published within 1 year
- [ ] Sources older than 1 year marked with "warning: potentially outdated"

## 4. Bias and Counter-evidence
- [ ] 3 sources are not biased toward a single viewpoint
- [ ] At least 1 opposing/critical perspective investigated

## 5. Relevance
- [ ] Each source's relevance to the idea scores >= 5/10

## On Failure
- Check 1-2 fail → return to Phase C for additional search (max 2 retries)
- Check 3-5 fail → replace only the failing sources
- After 2 retries still failing → report honestly to user, request manual judgment
